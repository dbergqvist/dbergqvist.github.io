# dbergqvist.github.io

Personal blog and portfolio site built with Hugo and the PaperMod theme.

## About

This is the source code for my personal website hosted on GitHub Pages.

## Tech Stack

- **Static Site Generator**: Hugo
- **Theme**: PaperMod
- **Hosting**: GitHub Pages

## Development

To run locally:

```bash
hugo server -D
```

To build for production:

```bash
hugo --minify
```
